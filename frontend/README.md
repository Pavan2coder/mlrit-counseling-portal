# 🎓 MLRIT Student Portal - Digital Counseling Book

A full-stack, mobile-responsive web application designed to digitize the student counseling and academic tracking process at MLR Institute of Technology (MLRIT). Built with the modern MERN stack and an ultra-clean "Glassmorphism" UI.

## ✨ Key Features
* **Secure Authentication:** Custom login and registration system using college email and Roll Numbers.
* **Modern UI/UX:** High-end "Glassmorphism" design using Tailwind CSS, featuring MLRIT's official branding (Green & Orange).
* **Mobile-First Design:** Fully responsive dashboard that adapts seamlessly to desktop and smartphone screens.
* **Interactive Dashboard:** Personalized student overview highlighting UGC Autonomous status and NBA Tier-1 accreditations.
* **Digital Counseling Records:** A dynamic self-reporting system where students can log and track their faculty interactions and academic remarks.
* **Secure Database:** Persistent data storage using MongoDB Atlas.

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite, Tailwind CSS, Framer Motion, Lucide React (Icons)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (NoSQL)
* **API Communication:** Axios
* **Notifications:** React Hot Toast
